## Customization Options

### 1. Header Alignment
- **Left**: All header content aligned to the left
- **Center**: All header content centered (default)
- **Right**: All header content aligned to the right

### 2. Logo Position
- **Left**: Logo positioned on the left side
- **Center**: Logo centered (default)
- **Right**: Logo positioned on the right side

### 3. Font Sizes
- **Small**: Compact layout, more information per page
- **Medium**: Balanced readability (default)
- **Large**: Enhanced readability, larger text

### 4. Border Options
- **With Border**: Full border around invoice (default)
- **Without Border**: Clean, borderless design

## Common Invoice Layouts

### Layout 1: Professional (Default)
- Centered header with logo at top
- Medium font size
- Full border
- All clinic information displayed

### Layout 2: Minimal
- Left-aligned header
- Logo on left
- Small font size
- No border
- Essential information only

### Layout 3: Modern
- Centered header
- Large logo at top center
- Large font size
- No border
- Clean, spacious design

## Prescription Format

The notes/prescription section can include:
- Diagnosis
- Treatment plan
- Exercise recommendations
- Follow-up schedule
- Precautions
- Medication (if authorized)

Example:
```
Diagnosis: Lower back pain (Lumbar strain)

Treatment Plan:
- Manual therapy sessions (3x per week)
- Core strengthening exercises
- Heat therapy

Exercises:
1. Pelvic tilts - 10 reps, 3 sets
2. Knee to chest stretch - Hold 30 seconds each side
3. Cat-cow stretch - 10 reps

Follow-up: After 1 week
Precautions: Avoid heavy lifting
```

## Tips for Invoice Customization

1. **Logo Guidelines**:
   - Use PNG or JPG format
   - Recommended size: 300x200 pixels
   - Keep file size under 500KB
   - Use transparent background for best results

2. **Clinic Information**:
   - Include all contact details
   - Add registration number for credibility
   - Include GST number if applicable
   - Add operating hours if desired

3. **Professional Touch**:
   - Use consistent formatting
   - Keep language professional
   - Include clinic tagline if any
   - Add clinic website if available

4. **Legal Requirements**:
   - Include doctor's registration number
   - Add GST details if registered
   - Include terms and conditions if needed
   - Add refund policy if applicable
