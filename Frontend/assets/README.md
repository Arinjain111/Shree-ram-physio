# Assets Folder

This folder should contain:

1. **icon.png** - Application icon (256x256 px recommended)
2. **icon.ico** - Windows icon file
3. **logo.png** - Default clinic logo (optional)

## Creating an Icon

You can create a simple icon using online tools:
- https://favicon.io/
- https://www.icoconverter.com/

Or use any image editing software like:
- GIMP (Free)
- Photoshop
- Paint.NET

## Recommended Icon Specifications

- Format: PNG and ICO
- Size: 256x256 pixels (PNG), Multiple sizes in ICO
- Background: Transparent
- Style: Simple, recognizable, professional

## Placeholder

Until you create your custom icon, the application will work without it.
Simply add your icon files here when ready:
- icon.png
- icon.ico

## Logo Guidelines

For best results with clinic logos:
- Use PNG format with transparent background
- Recommended size: 300x200 pixels
- Keep file size under 500KB
- Ensure good contrast against white background
